This directory contains datasets for use in the Prison HIV Project.

ICPSR_34540 contains the manifest, literature, and terms of use. 
The /DS0001subdirectory contains the datasets in .tsv and .rda form along 
with the codebook, documentation, and questionnaire forms.

The dataset is freely available at:

http://www.icpsr.umich.edu/icpsrweb/NACJD/studies/34540/version/1
